
Используется
=====================
### Create React App 
* PropTypes 
* Axios 
* Radium 
* Redux
* БД firebase.google.com

### Описание. 
 Адаптивное приложение для заказа Бургеров. 
 Имеется страница выбора ингридиентов, страница отправки заказа и страница выполненных заказов.
 Заказы сохраняются в firebase.google.com
